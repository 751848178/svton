# 项目概览

> Svton 是一个基于 TypeScript 全栈技术的社区平台项目，采用 Monorepo 架构统一管理前后端代码。

---

## 🎯 项目定位

**社区助手** 是一个面向社区居民的生活互助平台，包含：

- **移动端小程序** - 居民使用入口
- **管理后台** - 运营管理入口
- **后端 API** - 统一数据服务

## 🎨 共享包设计

共享代码分为两类：

**公共包**（使用 `@svton` 命名空间，发布到 npm）：

```typescript
// API 调用
import { apiAsync } from '@svton/api-client';

// React Hooks
import { usePersistFn, useDebounce } from '@svton/hooks';

// Taro UI 组件
import { NavBar, StatusBar, Button } from '@svton/taro-ui';
```

**私有包**（使用项目组织名，如 `@my-project`）：

```typescript
// 类型定义（项目私有包）
import type { ContentVo, UserVo } from '@my-project/types';
```
