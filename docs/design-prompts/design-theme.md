> 版本: 1.0.0

## 🎨 主题概述

这是社区助手小程序的统一设计主题，定义了颜色、字体、组件等基础样式规范。设计师可以在此基础上自由发挥创意，但建议保持整体一致性。

## 📝 文本样式

### 字体家族

```
默认字体:
- iOS: -apple-system, SF Pro Text
- Android: Roboto
- 通用: sans-serif

数字字体:
- SF Pro Display (用于统计数字)
- DIN (备选)
```

### 字号层级

```
特大标题: 36px (H1, 品牌名称)
大标题: 32px (H2, 页面标题)
标题: 30px (H3, 卡片标题)
正文大: 28px (重要正文)
正文: 26px (普通正文)
正文小: 24px (次要信息)
辅助文字: 22px (最小字号)
```

### 字重

```
Bold: 600 (标题、强调)
Medium: 500 (次级标题)
Regular: 400 (正文)
Light: 300 (辅助，慎用)
```

### 行高

```
标题行高: 1.2 - 1.4
正文行高: 1.6 - 1.8
辅助行高: 1.5
```

## 🧩 组件规范

### 按钮

#### 主要按钮

```
用途: 主要操作（登录、发布、提交等）
建议样式:
- 背景: 品牌蓝渐变或纯色
- 文字: 白色，字重600
- 圆角: 8-16px
- 高度: 44-52px
- 阴影: 可选，0 4px 12px rgba(24,144,255,0.3)
- 点击: 缩放95%或背景加深
```

#### 次要按钮

```
用途: 次要操作（取消、保存草稿等）
建议样式:
- 背景: 灰色或白色
- 文字: 深色，字重400-500
- 边框: 可选
- 圆角: 与主按钮一致
- 高度: 40-48px
```

#### 文字按钮

```
用途: 轻量级操作（更多、查看全部等）
建议样式:
- 无背景
- 文字: 品牌蓝或深灰
- 下划线: 可选
```

### 输入框

#### 文本输入

```
建议样式:
- 边框: 1-2px solid #E8E8E8
- 圆角: 8-16px
- 高度: 40-48px
- 内边距: 12-16px
- 占位符颜色: #AAAAAA
- 聚焦态: 边框变为品牌蓝
```

#### 多行文本

```
建议样式:
- 最小高度: 100-200px
- 内边距: 16px
- 行高: 1.6
- 其他同单行输入框
```

### 卡片

#### 内容卡片

```
用途: 展示内容、文章、动态等
建议元素:
- 封面图（可选）
- 标题
- 摘要/正文
- 作者信息
- 标签
- 互动数据（浏览、点赞、评论）
- 时间

建议样式:
- 背景: 白色
- 圆角: 12-20px
- 阴影: 可选，0 2px 8px rgba(0,0,0,0.04)
- 内边距: 12-20px
```

#### 信息卡片

```
用途: 展示用户信息、统计数据等
建议元素:
- 头像
- 名称/标题
- 描述/统计
- 操作按钮（可选）

建议样式:
- 背景: 白色
- 圆角: 8-16px
```

### 标签

```
用途: 分类、标记、筛选
建议样式:
- 高度: 20-28px
- 内边距: 6-12px
- 背景: 浅色（如 #F0F9FF）
- 文字: 对应深色（如 #1890FF）
- 圆角: 全圆角或4-8px
- 字号: 20-24px
```

### 头像

```
建议规格:
- 小: 32px
- 中: 48px
- 大: 96px
- 特大: 120px

建议样式:
- 形状: 圆形
- 边框: 可选，1-2px浅色
- 默认头像: 灰色占位
```

### 图标

```
建议规格:
- 小: 16px (装饰性)
- 中: 24px (常规)
- 大: 32px (强调)
- 特大: 48-64px (Logo、空状态)

建议来源:
- Emoji (简单场景)
- SF Symbols (iOS风格)
- Lucide Icons (现代简约)
- 自定义矢量图标
```

### 分割线

```
建议样式:
- 粗细: 1px
- 颜色: #F0F0F0 或 #E8E8E8
- 样式: 实线或虚线
- 间距: 8-24px
```

## 🎭 交互规范

### 点击反馈

```
推荐方式:
1. 缩放: scale(0.95-0.98)
2. 背景变化: 颜色加深或变浅
3. 透明度: opacity(0.7-0.8)
4. 阴影: 阴影加深或减浅

过渡时间: 200-300ms
缓动函数: ease-out 或 cubic-bezier
```

### 过渡动画

```
淡入淡出: 300ms ease-in-out
滑动: 300ms cubic-bezier(0.4, 0, 0.2, 1)
缩放: 200ms ease-out
弹跳: 400ms cubic-bezier(0.68, -0.55, 0.265, 1.55)
```

### 状态变化

```
加载: 显示骨架屏、加载动画或进度条
成功: 内容淡入，可选成功提示（1-2秒）
失败: 显示错误提示和重试按钮
空状态: 显示友好的插图和文字
```

## 📐 间距系统

### 基础间距单位

```
建议使用4px或8px为基础单位

常用间距:
xs: 4px   (紧凑间距)
sm: 8px   (小间距)
md: 16px  (中等间距，最常用)
lg: 24px  (大间距)
xl: 32px  (特大间距)
xxl: 48px (超大间距)

内容区边距:
手机端: 12-20px
平板端: 24-40px
```

### 网格系统

```
建议: 基于8px或12px网格
- 所有元素尺寸应为网格倍数
- 间距应为网格倍数
- 有助于保持视觉节奏
```

## 🌓 圆角系统

```
无圆角: 0px (特殊场景)
小圆角: 4-8px (小元素)
中圆角: 12-16px (卡片、输入框)
大圆角: 20-24px (大卡片、弹窗)
全圆角: 9999px 或 50% (按钮、标签、头像)
```

## 🎯 设计原则

### 一致性

```
- 相同功能的元素保持一致的样式
- 相同层级的文字使用一致的字号和颜色
- 间距遵循统一的网格系统
```

### 可访问性

```
- 最小字号不低于22px
- 最小点击区域不低于44×44px
- 颜色对比度符合WCAG AA标准
- 提供清晰的视觉反馈
```

### 性能

```
- 图片使用适当压缩
- 动画使用GPU加速（transform, opacity）
- 避免过度复杂的阴影和渐变
- 长列表使用虚拟滚动
```

### 情感化

```
品牌调性: 温馨、友好、可靠、现代
- 使用柔和的圆角
- 舒适的配色
- 适度的留白
- 友好的插图和文案
```

## 🔧 使用说明

### 给设计师

```
1. 这个主题规范是"建议性"的，不是强制的
2. 可以基于此主题自由创作，保持整体一致即可
3. 如需调整主题，请与团队讨论后更新此文档
4. 特殊页面可以有特殊的视觉风格，但核心元素保持一致
```

### 给AI设计师

```
使用此主题作为设计基础，但可以：
- 调整具体的圆角大小
- 优化间距和布局
- 创新组件样式
- 增加视觉细节
- 尝试不同的排版方式

核心约束：
- 品牌色保持#1890FF
- 字号不低于22px
- 点击区域不低于44px
- 保持整体风格统一
```

## 📦 导出资源

### 切图规范

```
格式: PNG (透明) / JPG (照片)
倍率: @2x, @3x
命名: icon_name@2x.png
压缩: TinyPNG 或 ImageOptim
```

### 图标库

```
推荐使用:
- SVG格式（可缩放）
- 单色图标（可换色）
- 24×24px基准尺寸
```

### 字体

```
iOS: 系统自带，无需导出
Android: 考虑使用思源黑体或Roboto
Web: 使用Web Font（如有需要）
```
