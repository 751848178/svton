## 🎯 项目概述

### 产品定位

社区助手是一款面向社区居民的生活服务小程序，提供邻里互助、信息分享、活动组织等功能。

### 目标用户

- **主要用户**: 25-45岁社区居民
- **使用场景**: 日常生活服务、邻里互动、信息获取
- **用户特点**: 注重实用性、简单易用、温馨友好

### 设计目标

1. **简约现代**: 去除冗余装饰，聚焦核心功能
2. **温暖友好**: 营造邻里间的温馨氛围
3. **易于理解**: 降低学习成本，直观易用
4. **视觉统一**: 保持各页面风格一致

1. 同一组件使用统一圆角

4. 大面积使用中等圆角

## 🎭 阴影系统

### 阴影层级

#### Level 0 - 无阴影

```
使用场景: 平面元素、分割线
```

#### Level 1 - 轻微阴影

```
box-shadow: 0 2px 8px rgba(0, 0, 0, 0.04)
使用场景: 卡片、列表项
```

#### Level 2 - 标准阴影

```
box-shadow: 0 2px 12px rgba(0, 0, 0, 0.08)
使用场景: 悬浮卡片、弹出框
```

#### Level 3 - 强调阴影

```
box-shadow: 0 4px 16px rgba(0, 0, 0, 0.12)
使用场景: 模态框、重要提示
```

#### Level 4 - 按钮阴影

```
box-shadow: 0 4px 12px rgba(24, 144, 255, 0.3)
使用场景: 主按钮、CTA按钮
```

### 阴影使用原则

1. 越重要的元素阴影越强
2. 层级关系通过阴影体现
3. 避免过度使用阴影
4. 品牌色按钮使用品牌色阴影

## 🎯 设计原则

### 1. 少即是多

去除不必要的装饰，让内容成为焦点。

### 2. 一致性优先

保持视觉风格、交互方式的统一性。

### 3. 用户至上

以用户需求为中心，降低使用成本。

### 4. 适度动效

恰到好处的动画让体验更流畅。

### 5. 温暖友好

通过色彩、圆角、间距营造温馨氛围。
